매일 공부한 단어를 복습하는 겸 남기는 메모장입니다.

| 단어 | 뜻 | 예문 | 
| pollute | 오염시키다 |  |
| pollution | 오염, 공해 |  |
| possible | 가능한 |  |
| practical | 실현 가능한, 실행 가능한 |  |
| pristine | 자연 그대로의, 오염되지 않은, 아주 깨끗한 |  |
| questionable | 의문의 여지가 있는, 미심쩍은 |  |
| recyclable | 재활용 가능한 |  |
| recycle | 재활용하다 |  |
| refuse | 폐기물, 음식물 쓰레기 |  |
